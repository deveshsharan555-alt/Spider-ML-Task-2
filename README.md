# Spider Inductions 2026 – Machine Learning Tasks

This repository contains the projects completed as part of the **Spider Inductions 2026** Machine Learning track.

======> Tasks

### Base Task

**Level 1 – Custom ResNet Image Classification**
- Built a ResNet model from scratch using PyTorch
- Trained and evaluated on the CIFAR-10 dataset

**Level 2 – Weather Forecasting using Manual LSTM**
- Implemented an LSTM from scratch (without `nn.LSTM`)
- Trained on the Jena Climate dataset for temperature forecasting

**Level 3 – Weather Forecasting using Transformer**
- Built an encoder-only Transformer for multi-step weather forecasting
- Compared its performance with the LSTM model

### Applied ML Task

**Healthcare Information Assistant (RAG)**
- Developed a Retrieval-Augmented Generation (RAG) based healthcare assistant
- Uses FAISS for document retrieval and Streamlit for the user interface
- Provides grounded responses with source citations
